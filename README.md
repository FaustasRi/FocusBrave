# FocusBrave

FocusBrave is a macOS focus application built with SwiftUI.

## Features

• Focus timer  
• App blocking during focus sessions  
• Website blocking via /etc/hosts  
• Menu bar controls  
• Persistent blocked apps & websites  

## Blocked Apps

Brave  
Chrome  
Safari  
Discord  

## Blocked Websites

YouTube  
Reddit  
X / Twitter  

## Architecture

SwiftUI  
AppKit integration  
macOS process monitoring  
Hosts-based web blocking
